Logo Usage — Do & Don't
✓ Do
  • Use the Blueprint Grid icon + wordmark lockup as the primary logo
  • Use the white wordmark + orange "NEW CONSTRUCTION" on dark backgrounds
  • Use the dark wordmark + orange subline on white backgrounds
  • Use Barlow Condensed exclusively for the logo wordmark
  • Maintain minimum clear space equal to the icon height on all sides
  • Use Bebas Neue for all display and section headings
✗ Don't
  • Stretch, skew, or rotate the logo
  • Change the icon stroke or fill color to anything other than brand colors
  • Use the logo on busy photographic backgrounds without a dark scrim
  • Use Barlow Condensed for general headings — logo wordmark use only
  • Use Inter for display headlines — always use Bebas Neue
  • Use accent orange for body text
